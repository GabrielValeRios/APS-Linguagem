# APS-Linguagem
Tópicos "WishList":
  - Há ";".
  - Loops e Condições exigem "{}" ao começo e final. Ex: " if (x>4){do something}.
  - "Else" será mantido.
  - Não existe "For".
  - Os programas irão rodar em um "Main". Ex: " Main(): blablabla"
  - Comandos serão em inglês.
  - A linguagem terá suporte para println.
